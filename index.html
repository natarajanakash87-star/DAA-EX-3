Implementation of Kruskal's and Prim's Algorithms for MST

This project implements both Kruskal's and Prim's algorithms for finding the Minimum Spanning Tree of a weighted undirected graph containing seven vertices and nine edges.

Conclusion

Both Kruskal's and Prim's algorithms successfully produce the same minimum spanning tree weight of 12. Kruskal's algorithm is edge-oriented whereas Prim's algorithm is vertex-oriented. Their efficiencies depend upon the graph representation and data structures used.
